*The things I use to get work done. No affiliate links. No "sponsored" section pretending to be a recommendation.*

---

**Editor**

VS Code. I've tried the others. I came back.

**Terminal**

Whatever ships with the OS, plus a font that makes monospace legible. Paying $15/month for a terminal is not something I do.

**Browser**

Firefox. Chromium when Firefox doesn't work. Chrome never.

**Hosting**

Static files on a CDN. If your website needs a server running 24/7, you've overcomplicated your website.

**Writing**

Plain text files with Markdown. A folder called `notes`. No Notion, no Obsidian, no "second brain" framework.

**Hardware**

A laptop. A monitor. A keyboard. You don't need more than this.
